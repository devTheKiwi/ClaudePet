ClaudePet

Dock 위를 돌아다니는 Claude 데스크탑 펫. 작업 상태를 알려주고, 가끔 말도 걸어줍니다.

설치하기 GitHub

주요 기능

🔗

Claude Code 실시간 연동

Hook 기반으로 작업 시작, 완료, 권한 요청을 즉시 감지합니다.

☕

Claude Desktop 지원

Desktop 앱 켜면 커피잔 든 펫이 등장! 사용 시간도 추적합니다.

🪙

토큰 사용량 추적

세션/일일 토큰, 캐시 절약률, 마일스톤 알림까지.

🔧

도구별 상세 알림

"명령어 실행 중...", "코드 수정 중..." 도구마다 다른 말풍선.

⏱

작업시간 트래커

실제 작업 시간을 초 단위로 추적. 우클릭으로 세션/오늘 통계 확인.

🎨

스킨 시스템

봄 에디션 벚꽃, 시즌별 스킨. 우클릭으로 변경 가능.

👆

상호작용

클릭하면 반응, 더블클릭하면 점프, 우클릭으로 상태 확인.

🔄

자동 업데이트

새 버전 나오면 Pet이 알려줌. PC 시작 시 자동 실행.

스킨

우클릭 > 스킨에서 변경할 수 있어요
